# Fraction — Арифметика дробей с произвольной точностью

Программа на C++ для работы с обыкновенными дробями с использованием библиотеки **GMP** (GNU Multiple Precision Arithmetic Library). Поддерживает числители и знаменатели произвольного размера.

---

## Возможности

- Представление дробей в виде `числитель / знаменатель`
- Автоматическое сокращение дробей (через НОД)
- Нормализация знака (знак всегда у числителя)
- Арифметические операции: сложение, вычитание, умножение, деление
- Ввод чисел произвольной точности через GMP
- Защита от деления на ноль

---

## Зависимости

| Зависимость | Версия | Установка |
|---|---|---|
| GCC / G++ | ≥ 9 | `sudo apt install g++` |
| GMP | ≥ 6.x | `sudo apt install libgmp-dev` |

---

## Сборка

```bash
g++ -o fraction main.cpp -lgmp
```

---

## Запуск

```bash
./fraction
```

Программа запросит числитель и знаменатель для каждой из трёх дробей:

```
Введите три дроби

Дробь 1:
  Числитель:   1
  Знаменатель: 2
  Десятичный вид: 0.5

Дробь 2:
  Числитель:   3
  Знаменатель: 2
  Десятичный вид: 1.5

Дробь 3:
  Числитель:   4
  Знаменатель: 2
  Десятичный вид: 2

Результат (обыкновенная дробь): -3 / 1
Результат (десятичная): -3
Дробь успешно обращена!
После обращения (обыкновенная): 1 / -3
После обращения (десятичная): -0.33333333333333333333
```

Программа вычисляет: `(дробь1 + дробь2) * (дробь1 - дробь3)`

---

## Структура кода

### Класс `Fraction`

| Метод | Описание |
|---|---|
| `Fraction()` | Конструктор по умолчанию, создаёт дробь `0/1` |
| `Fraction(n, d)` | Конструктор из числителя и знаменателя (mpz_t) |
| `operator=` | Оператор присваивания |
| `operator+` | Сложение дробей |
| `operator-` | Вычитание дробей |
| `operator*` | Умножение дробей |
| `operator/` | Деление дробей |
| `print()` | Вывод дроби в формате `a / b` |
| `normalize()` | Внутренний метод: сокращение и нормализация знака |
